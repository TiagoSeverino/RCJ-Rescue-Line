# Hardware

Processing Units

* 1x Picaxe 40x2
* 1x Picaxe 20x2
* 1x Picaxe 08M

Sensors

* 2x SRF04
* 8x Line Sensor
* 2x Silver(LDR) Sensor
* 2x RGB Sensor
* 7x Micro-Switch

Actuators

* 4x Motors
* 4x Servo Motors
